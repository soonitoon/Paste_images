# Paste_images
The Paste_images is a program that can make pasted long image.

## you can...
- add images to paste
- delete imagee you dont want to paste
- choose save directory

## options
- image size(raw,0 1024, 800, 640)
- image-to-image distance(none, narrow, normal, wide)
- fomat(PNG, JPG, BMP)
